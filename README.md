# Coffee-Sales-Dashboard

![](https://github.com/MarvelousOtis/Coffee-Sales-Dashboard/blob/main/titleimage.jpg)

## Introduction:

This project analyzes coffee sales performance over time, across countries, roast types, package sizes, and customer loyalty status. 
The aim is to understand customer preferences and sales trends.

## Key Metrics:
-	How do coffee sales change over time?
-	Which countries generate the most revenue?
-	Which customers contribute the most to sales?
-	How do roast types and package sizes affect sales?
-	Does having a loyalty card impact purchasing behavior?

## Skills & Concepts Demonstrated:
-	Time-based sales analysis
-	Customer segmentation
-	Comparative analysis across regions
-	Use of slicers for filtering by roast type, size, and loyalty card
-	Visual storytelling with line and bar charts
